> This is the short alias for **[`bagidea-office`](https://www.npmjs.com/package/bagidea-office)** —
> `npx bagidea` and `npx bagidea-office` do the same thing.

## Install

```sh
npx bagidea
```

Runs the official installer for your platform — it **downloads a prebuilt app**, so a
normal install needs **no Visual Studio Build Tools or Rust**. After install, manage the
office with the **`bagidea`** command (`start` / `stop` / `restart` / `update`).

- **Windows** — fully supported · **macOS / Linux** — beta
- Needs [Claude Code](https://claude.com/claude-code). Model brains are swappable per agent
  (Claude, GLM, DeepSeek, Qwen, OpenAI, Gemini, or local via Ollama / LM Studio).

> **Truth, not theatre.** A glowing screen is a live tool call. A walk to "Security" is a
> real permission request. If it's on screen, it's really happening.
